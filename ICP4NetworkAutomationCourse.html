Cloud Pak for Network Automation 2.1 Administration and Configuration

Information on this course

This course is a paid course. Please contact Tom Reed for how to pay for the course and get enrolled.

This course dates listed below are guaranteed to run as long as there is a paying student enrolled.

The ZN777 course is a self-paced with lab course. The course will take two days to complete. The lectures are recorded and then the labs can be completed after each lecture.

You will be given three days to complete the labs for this course. Be certain you allow sufficient time to commplete the labs. There are no extensions to the lab environment.

The free prerequisite for this course is DL10032G. The prerequisite course should be completed before starting the ZN777 course.

This course is intended to teach the skills that are needed to successfully install, configure, and use Cloud Pak for Network Automation (CP4NA) 2.1. This course is the second part of instructor-led or self-paced training in Cloud Pak for Network Automation and focuses on hands-on lab exercises to describe, deploy, and manage VOIP and telco components. The first portion of the course is DL10032G that focuses on the fundamental principles, architecture, and terminology of CP4NA.
